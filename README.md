# GitHub Profile Analyzer

A clean and interactive web application that allows users to search for GitHub profiles and view their key statistics and latest repositories.

## Features

- **Profile Overview:** Displays user's avatar, name, and bio.
- **Key Statistics:** Shows the number of followers, following, and public repositories.
- **Latest Repositories:** Lists the 5 most recently updated public repositories with links and descriptions.
- **Real-time Data:** Fetches information directly from the GitHub REST API.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## Technologies Used

- HTML5
- CSS3 (Custom styling)
- JavaScript (ES6+, Fetch API)
- GitHub REST API

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/tharunguna572006-prog/github-profile-analyzer.git
   cd github-profile-analyzer
   ```

2. Open `index.html` in your web browser.

3. Enter a GitHub username in the search box and click "Search".

## Project Structure

- `index.html`: The main HTML structure of the application.
- `styles.css`: Custom CSS for styling and layout.
- `script.js`: JavaScript logic for fetching data and updating the UI.
